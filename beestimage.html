Responsive image + context

The result of this process

This process will generate a responsive image element that is side by side with the context for what students can expect to know/see with the image. A responsive image changes size with the size of the browser window you are in and when the screen gets small enough, the image and text in this combo will 'collapse' underneath one another. This tool will change your image link into the embed link for the respective platform. This tool currently accepts the following image links - Unsplash, Pexels, StockSnap. Click on the links below to reveal which link you require for the BEEST responsive image tool Step 1. If you click play on the video and exit the overlay, the video will continue to play.
