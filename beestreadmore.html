Monash themed read more buttom

This process creates a read more button that can hide extended text, and reveal or collapse when clicking the button.

Important caveats

Overuse of these read more buttons, or putting to much content behind the toggle can result in poor user experience. As a guide, you should only hide between 15-20 lines of text behind the read more button.

The result

The end result here is the a read more button that hides extended text that reveals or collapses on click.

Step 1 - What number read more button on your page is this before your read more button.

If you want to display on read more button on a page, simply put the number 1 in the box below. If you want to put more than one read more button on a page, you need to number them so that they work. For example. If you make two read more buttons that are both number one, clicking one of the buttons will result in only the first read more button opening and closing the full text.

Using more than 3 read more buttons on a page could result in poor useability for students.

Step 2 - What are you showing/hiding?

The read more button will always start with Show/hide. What you put in the text box below will appear after Show/hide. For example Show/hide full bio or Show/hide full text.
