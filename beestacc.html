You can only have one accordion on a page because of the way the HTML works. Please make sure you only produce, copy and paste one accordion per Moodle page maximum. If you try to put two or more accordions on a page, none of them will work correctly.

Accordion

The result of this process

This process will generate an accordion that can display information collapsed under heading buttons (drawers). It's important to note that accordions can easily be misused.

  • They should be used to contain information such as frequently asked questions (FAQs), or putting together important unit information such a Synopsis, Learning Outcomes, Assessment Summary all in the one place.
  • If they get used on every single page they end up frustrating users who constantly have to click to reveal information.
  • if you put too much information in the drawers, they will open and the clickable heading will disappear off the page. Keep it to 5-8 lines of information wherever possible.

How to use this page

If you hover your mouse over the headings for the steps on this page instructions will appear. Once you have generated and copied the code for your element paste them into Moodle using the HTML editor. These text editors differ slightly from the other BEEST pages. They allow you to input HTML. So if you want to embed a video or other external content, or even content from another BEEST page (video, alert, button) you can. If you are unsure of how to paste HTML into these text editors or Moodle, this short video will help (it will open in an overlay on this screen).
