BEEST Project

BEEST stands for Bootstrap Enhanced Elements Streamlining Tool. Bootstrap is the worldâ€™s most popular front-end open source toolkit, featuring Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful JavaScript plugins This site combines pre-designed HTML code elements with user input to create HTML elements to enhance Moodle site formatting and user experience. Please click on the buttons below to be taken to the page where you can design your desired element. All buttons open in this window and examples of each of the element types can be viewed on each page.

Important caveats

  • These elements require Bootstrap v4.0 and up to function. The BEEST pages here are purpose built for use with Moodle 3.7 and upwards to manipulate content to be responsive and inviting to the user.
  • Some elements also require a stylesheet reference to function correctly. Where needed this reference will appear so that you can copy and paste it along with your element to ensure it works correctly.
