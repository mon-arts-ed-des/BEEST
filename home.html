BEEST Project

What is BEEST?

BEEST stands for Bootstrap Enhanced Elements Streamlining Tool. This site combines pre-designed HTML code with simple user input to create an engaging, responsive user experience (UX) in your Moodle site.

Why would I use BEEST?

Assessment, teaching and learning design that builds an interactive, meaningful, applied learning experience for students is the key to teaching and learning success for both the teacher and the student. As technology and the digital learning space evolves at a rapid pace, learners are also beginning to expect a seamless and intuitive user experience from Moodle site to Moodle site, and across all device types. But building responsive, consistent user experiences requires time, effort, a basic understanding of UX and restraint in using design. BEEST can help you achieve a consistent design and cut down on time and effort required to build complex elements needed for an engaging UX across all your Moodle sites, you just need to bring the content, consistency and constraint.

What can BEEST do that nothing else can?

BEEST, when used appropriately, can help you build a consistent, well-formatted, responsive user experience in your Moodle site without the need for advanced coding skills.

How does BEEST make designing advanced elements easy?

The processes in this site combine pre-built HTML with your input. The user only needs to know how to follow instructions on each page to find the correct links from different platforms for the BEEST element you are creating, copy and paste text and links, use rich text formatting like you would in Moodle and also how to paste HTML into the Moodle editor.

Important caveats

  • These elements require Bootstrap v4.0 and up to function. The BEEST pages here are purpose built for use with Moodle 3.7 and upwards to manipulate content to be responsive and inviting to the user.
  • Some elements (the Monash blue elements) also require a stylesheet reference to function correctly as this is not part of the Bootstrap styling guide.
