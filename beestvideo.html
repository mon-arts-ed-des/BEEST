Responsive video + context

The result of this process

This process will generate a video iframe that is responsive that is side by side with the context for what students can expect to see in the video. A responsive video changes size with the size of the browser window you are in and when the screen gets small enough, the video and text in this combo will 'collapse' underneath one another. This tool will change your video link into the embed link for the respective platform. This tool currently accepts the following video links - Panopto videos/playlists, Youtube, Vimeo, TED talks.
