Overuse of alerts on a page can become like OHS signage. They simply fade into the background, and all of a sudden the 'important' information you want students to notice, becomes less important than the rest of the text on the screen. Use them sparingly.

Alerts

The result of this process

This process will generate what is termed an 'alert'. It is a box that 'sticks out' from the rest of the text to alert students to important information on the page. You can choose whether to make this alert permanent, or add the option to dismiss/close it.

How to use this page

Once you have generated and copied the code for your element paste them into Moodle using the HTML editor. If you are unsure of how to paste HTML into Moodle, this short video will help (it will open in an overlay on this screen).
