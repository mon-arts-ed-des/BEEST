Monash button - opens in a new window

This process will generate a Monash blue button element with text that communicates where the button links to and the link will open in a new window. This is recommended for content you are linking to outside of Moodle, so that Moodle stays open in a separate tab for students. Please ensure the links you paste are full URLs.

E.g. https://google.com or https://lms.monash.edu.

If you do not include the https:// or http:// prefix your button will not work correctly.

The result

The end result will be similar to the below example (your link and text will be different). We've included Font Awesome in this example (the arrow pointing to the right) which will be added to the end of your text.

Go to Monash Moodle

Step 1 - Paste your URL

Paste in the link/URL for the button. Where users will be directed to when clicking the button.

Step 2 - Write the text for the button

You can have text that tells the user what the button is linked to here - recommended less than 25 characters including spaces).

Step 3 - Open your link in the same window or a new window

You should open your link in a new window if it is opening a site outside of Moodle or resource you want students to read/view. If you are linking to another part of the same Moodle site, e.g. a forum, then open the link in the same window. New window is chosen by default

Step 4 - Generate and copy the code for your element

When you click the button below, it will generate the HTML for your element and copy it, ready to be pasted into Moodle. If you are unsure of how to paste HTML into Moodle watch this short video here.
